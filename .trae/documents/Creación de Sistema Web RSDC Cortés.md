# Plan de Desarrollo: Sistema RSDC (Región Sanitaria Departamental de Cortés)

Este plan detalla la creación de una aplicación web moderna, futurista y funcional para la gestión de procesos de la Región Sanitaria de Cortés.

## 1. Configuración Inicial del Proyecto

* **Tecnologías**: Vite + React + TypeScript.

* **Estilos**: Tailwind CSS para un diseño responsivo y moderno.

* **Animaciones**: Framer Motion para transiciones suaves y efectos futuristas.

* **Iconografía**: Lucide React para iconos limpios y consistentes.

* **Base de Datos**: Integración con Supabase (Proyecto ID: `sb1-odmyggm7`).

## 2. Diseño Visual y UI/UX

* **Estética**: Estilo "Glassmorphism" con fondos semi-transparentes, desenfoque (backdrop-blur) y bordes sutiles.

* **Colores**: Paleta de colores oscuros con acentos en gradientes azul cian y violeta neón.

* **Componentes**: Botones con efectos de brillo, tarjetas con esquinas redondeadas y tipografía moderna (Inter o similar).

## 3. Estructura de Navegación y Layout

* **Navbar Superior**: Contendrá el botón de hamburguesa en la esquina izquierda.

* **Sidebar Desplegable**: Menú lateral que cubre el alto de la pantalla con las opciones:

  * Home

  * Números de Oficio

  * Documentos en Dirección

  * Registro de Vehículos

  * Ajustes

  * Términos y Condiciones

## 4. Desarrollo de Páginas y Funcionalidades

* **Autenticación**: Páginas de Login y Registro conectadas a Supabase Auth.

* **Home**: Dashboard con resumen de procesos y accesos directos.

* **Números de Oficio**: Módulo para la gestión y redacción de oficios.

* **Documentos en Dirección**: Seguimiento del estado de documentos entregados.

* **Registro de Vehículos**: Control de entradas/salidas y pases de salida.

* **Ajustes y Términos**: Configuración de usuario y avisos legales.

## 5. Integración con Supabase

* Configuración del cliente de Supabase para interactuar con la base de datos `sb1-odmyggm7`.
